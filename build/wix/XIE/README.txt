XIE 1.0
===

XIE は Eggs Imaging Elements の略称です。  
画像処理アプリケーション開発を補助することを目的としたライブラリです。  
データ構造、画像表示、画像取り込み、通信等を実装しています。  


## 説明

本ライブラリは C++ 及び .NET を対象にしています。  
使用方法について詳しくはリファレンスマニュアル(XIE-cpp.chm)をご参照ください。  
.NET 版のリファレンスについては別冊の XIE-cli.chm をご参照ください。  
現在のところ、リファレンスマニュアルをインストーラに含めていませんので  
別途ダウンロードしてください。  

注意）  
現在、バージョン 1.0 を開発中です。  
機能の追加や変更等により互換性が断たれる可能性があります。  
開発が収束まではバージョン更新を行いませんのでご注意ください。  
また、リファレンスマニュアルの内容も未記載や誤り等があります。
